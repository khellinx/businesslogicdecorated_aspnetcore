## Installation

To add the toolbox to a project, you add the package to the project.json :

``` json 
"dependencies": {
    "Digipolis.BusinessLogicDecorated":  "0.1.1"
 }
``` 

In Visual Studio you can also use the NuGet Package Manager to do this.
